# Safari Extension (Coming Soon)

Safari extensions require:
- macOS with Xcode
- Apple Developer account ($99/year)
- Swift/SwiftUI for native wrapper
- App Store review process

## Architecture

Safari uses a different extension model:
- Native macOS/iOS app wrapper
- JavaScript content scripts
- Native messaging between Swift and JS

## Development Notes

```
1. Create new Xcode project
2. Add Safari Extension target
3. Convert manifest.json to Info.plist
4. Implement SafariExtensionHandler
5. Port JavaScript to Safari content script format
```
