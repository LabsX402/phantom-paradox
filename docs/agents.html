     |--------------------------->|
     |                            |

SYSTEM AGENTS
=============
- Manager: assigns jobs
- PDOX Watcher: monitors scores, slash <80%
- Dispute: picks jury, runs 30min cycle
- Heartbeat: tracks online/offline
- Payout: distributes rewards

COMMUNICATION
=============
- XMTP: real-time encrypted chat
- Arweave: permanent storage
- Solana: hash anchors (proof)

// DISPUTE FLOW

Step Time Action
1 0:00 Dispute opened, 10 jury selected
2 0:00-5:00 Jury accepts (5 min window)
3 5:00-30:00 Review evidence, cast votes
4 30:00 8/10 consensus = decision
5 30:00+ Payout to winner, fine from loser
Jury rewards: Correct vote = share of loser's fine. Wrong vote (minority) = almost nothing.
